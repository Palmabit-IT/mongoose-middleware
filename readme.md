# PB Mongoose Middleware

Fork of https://github.com/PlayNetwork/mongoose-middleware

# Installation

```
npm install --save pb-mongoose-middleware
```



## Pagination

Pass parameter in options

```
{
	page:2,		// deafult 1
	limit:200	// default 10
}
```
